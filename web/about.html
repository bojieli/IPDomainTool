History

This website is a team project in ASE class.

We came to MSRA as interns in July, 2013. We had an Advanced Software Engineering and made a team to finish one team project together. After discussion, we decided to establish a web site with IP and domain searching service.

This project started in November, 2013, and the alpha version was released in December. Now this project is still in development and may be finished in January, 2014.
